連絡先クリーナー (Contact Cleaner)

スマホのアドレス帳（VCFファイル）の重複や表記ゆれをブラウザ上で自動整理し、きれいな状態のVCFファイルとしてエクスポートできるWebアプリケーションです。

✨ 主な機能

重複の自動統合: 同じ名前で登録されている連絡先を見つけ、電話番号やメールアドレスなどの情報を一つにまとめます。

電話番号のフォーマット統一: ハイフンなしや表記ゆれのある電話番号を、日本の標準的なフォーマット（例: 090-XXXX-XXXX）に統一します。

フリガナのひらがな変換: 半角カナや全角カタカナで登録されているフリガナ（SOUNDプロパティ）を、すべて「ひらがな」に自動変換します。

オフライン動作・プライバシー重視: データの処理はすべてブラウザ（クライアントサイド）で完結するため、サーバーに個人情報が送信されることはありません。

PWA対応: スマートフォンのホーム画面に追加して、ネイティブアプリのように利用できます。

🛠 技術スタック

フロントエンド: React 18 (CDN経由での利用)

スタイリング: Tailwind CSS

フォント: LINE Seed JP

アイコン: カスタムSVGアイコン

ビルドツール不要: index.html 1ファイル（CDNとBabel Standaloneを利用）で動作します。

🚀 使い方

1. VCFファイルで全件一括処理（推奨）

AndroidやiPhoneの連絡先アプリから全件をエクスポートし、本アプリで処理する方法です。

スマートフォンの「連絡先」アプリを開きます。

設定やメニューから「連絡先のエクスポート」を選択し、*.vcf (vCard) 形式でファイルに保存します。

本アプリを開き、「VCFファイルで全件一括処理」ボタンをタップします。

保存したVCFファイルを選択すると、自動的に整理が始まります。

完了画面が表示されたら、新しくダウンロードされたVCFファイルを連絡先アプリでインポート（復元）してください。

2. 連絡先を選んで整理（Android / 対応ブラウザのみ）

Contact Picker API をサポートしている環境（主にAndroidのChrome）では、端末の連絡先から特定のデータを選択して整理することができます。

本アプリを開き、「連絡先を選んで整理」ボタンをタップします。

端末の連絡先ピッカーが開くので、整理したい連絡先を選択します。

自動的に整理・結合され、新しいVCFファイルがダウンロードされます。

仕組み・仕様

VCFパーサー: 独自の正規表現ベースのパーサーを用いて vCard(BEGIN:VCARD ~ END:VCARD) データを解析します。

マージロジック: FN (フルネーム) または N (氏名) をキーとして同一人物を判定し、欠けている情報（電話番号の追加、メールアドレスの追加）を補完・統合します。

出力: VERSION:3.0 の vCard フォーマットで出力します。文字化け防止のため、出力ファイルには BOM (Byte Order Mark) を付与しています。

📱 インストール (PWA)

このアプリは PWA (Progressive Web App) としてインストール可能です。
対応ブラウザでアクセスすると、画面右下に「インストール」ボタンが表示されます（スタンドアロンモードで開いている場合を除く）。
